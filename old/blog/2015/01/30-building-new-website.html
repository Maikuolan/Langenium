Building new website



I don't even know how many times I've rebuilt the website now, although this is probably only the second presentable build.


My original plan was to build a fully dynamic site but the CSS took too long to load and it was off-putting. I attempted to find ways to create intermediate displays before the engine could boot up, but it just wasn't worth the effort.


The website is now completely made of hardcoded HTML. This has actually made my job a lot easier and I'm surprised I hadn't thought of this sooner. Code is simply served via a single framework hanging off the global "L" variable.
